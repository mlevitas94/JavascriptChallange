Purple Rain

Description

This module utilizes on click event listeners to render styled purple droplets to the target area as long as the mouse button is held down. Upon mouse up, the droplets are removed, allowing for a transition of an unencumbered DOM. CSS animations are used to transition the droplets downwards, originating from wherever the mouse is in the project area.

Instructions

Hold down the standard mouse click on the project area to start the animation. Stop holding the mouse button to stop animation.

New Years Count Down

Description

At every one second interval, Javascript calculates the the time difference between now and the next new year date. The difference is rendered into separate sections labeled : Days, Hours, Minutes, Seconds. A custom flipping animation is executed if there is a change in one of the time sections, rendering the next number behind the front elements and then resetting.

Drag and Drop

Description

DOM manipulation that allows you to re-locate elements from one area to another. Javascript is used to render the clicked item on the mouse and move it with the mouse’s position as well as checking to see where the item is placed when the mouse is un-clicked.

Instructions

You can add items to group one but typing something in the input field followed by pressing add. You can then click and drag the item over to group 2 and release the mouse button to place. You can drag items from group 2 to group 1 as well.

Hearthstone Card Search

Description

Javascript search engine that utilizes a 3rd party API (https://hearthstoneapi.com/) that serves data from the card game : Hearthstone. Upon entering this web page, it makes a request and loads all card data at once. You can then search through the cards based on certain properties.

Instructions

Pressing search will render four cards based on the selected properties on the left. You can press load more under the four cards to load the next four if there are any. If a property is not selected or is labeled N/A, it will ignore the parameter.

Tic Tac Toe

Description

A tic tac toe game you can play locally or online with another person in real time. This module utilized Socket.io to perform live updates to the DOM via Javascript.

Instructions

Press local to start a game between two people that are in person. Press Online to reveal more options to create an online game. Creating a room gives a room number which another person can enter when they join a room. Once the second person joins, the game starts. Once the game is over, players have the option to return or play again. If play again is chosen, the second player can press the same option to be reconnected for another game.
